# ERP Procurement Dashboard

## Overview

This project demonstrates procurement reporting and dashboard development using ERP procurement data.

The dashboard was designed to help procurement teams monitor purchasing activities, approval status distribution, buyer workload and procurement spending.

## Objectives

* Monitor procurement operations
* Identify approval bottlenecks
* Analyse buyer workload distribution
* Track procurement spending trends
* Support data-driven decision making

## Tools

* Oracle ERP
* SQL
* BigQuery
* Looker Studio

## Dashboard Metrics

* Total Purchase Orders
* Approval Status Distribution
* Buyer Workload
* Procurement Spend by Currency
* Unapproved Purchase Orders
